# face-detection

### Install pip
-
`
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
`
-
`
python get-pip.py
`

- python version
v3.10.8
- pip version
v22.2.2.2

### Environment on Windows
- Create Environment
`
python -m venv fdEnv
`

- Start Environment
`
fdEnv\Scripts\activate
`
